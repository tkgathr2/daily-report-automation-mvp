使い方ガイド

1. 日報の編集

各セクションで日報の内容を編集できます。ツールバーで太字・斜体・リストなどの書式を適用できます。

2. Slack連携

初回は「Slack連携（認可）」ボタンでSlackと連携してください。連携後は「Slackに投稿」ボタンで日報を送信できます。

3. ショートカットキー

Ctrl+B: 太字 / Ctrl+I: 斜体 / Ctrl+U: 下線 / Ctrl+Enter: Slackに送信

4. 下書き保存

入力内容は自動的に下書き保存されます。ページを閉じても、次回アクセス時に復元できます。
