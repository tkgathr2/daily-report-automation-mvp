## 確認手順

### Step 1: WEBアプリにアクセス

上記URLをブラウザで開いてください。

---

### Step 2: 表示確認

「取得ツール設定」エリアを確認してください。

**期待結果**:
```
取得ツール設定  ☑ Slack  ☑ Gmail  ☑ Notion  [設定を保存]
```

**確認ポイント**:
- [ ] **Slack** チェックボックスが表示されている
- [ ] **Slack** の文字が紫色（#4A154B）で表示されている
- [ ] **Gmail** チェックボックスが表示されている（既存）
- [ ] **Notion** チェックボックスが表示されている（既存）

---

### Step 3: 動作確認

1. **Slack** チェックボックスをオフにする
2. 「設定を保存」ボタンをクリック
3. 「設定を保存しました。」とメッセージが表示されることを確認
4. ページをリロード（F5）する
5. **Slack** チェックボックスがオフのままであることを確認
6. **Slack** チェックボックスをオンに戻す
7. 「設定を保存」ボタンをクリック

**確認ポイント**:
- [ ] 設定が正しく保存される
- [ ] ページリロード後も設定が維持される

---

### Step 4: 履歴取得確認（任意）

1. **Slack** チェックボックスをオンにする
2. 「設定を保存」をクリック
3. 「予定を取得」ボタンをクリック
4. 「今日やったこと」エリアに **[Slack]** 履歴が表示されることを確認
   （Slack連携済みで当日の投稿がある場合のみ）
