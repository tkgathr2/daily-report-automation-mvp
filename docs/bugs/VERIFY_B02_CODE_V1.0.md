## 動作確認が失敗した場合

### タスク1: 問題の切り分け

問題の内容に応じて調査：

1. **Slackチェックボックスが表示されない場合**
   - `src/Index.html` の993-999行を確認
   - `settingSlack` のIDが正しいか確認

2. **設定が保存されない場合**
   - `saveToolSettingsUI()` 関数を確認
   - `slack` プロパティが含まれているか確認

3. **リロード後に設定が維持されない場合**
   - `loadToolSettings()` 関数を確認
   - `settingSlack` の読み込み処理を確認

### タスク2: 修正と再デプロイ

問題を修正後：

```bash
clasp push
clasp deploy --description "B02 修正再デプロイ"
```
