# Devin タスク: V5 実装

## これは Devin への指示です。

---

## タスク概要

簡単日報くん V5（Zoom/Meet/社内ツール連携）を実装する。

## 前提条件

- V4が完成・稼働していること

## 参照ドキュメント

- 要件定義書: `docs/specs/V5_requirements_ASK.md`
- 仕様書: `docs/specs/V5_plan.md`

---

## 実装手順

### Step 1: Code.gs に定数追加

```javascript
const PROPERTY_ZOOM_TOKEN = 'ZOOM_ACCESS_TOKEN';
const PROPERTY_ZOOM_ACCOUNT_ID = 'ZOOM_ACCOUNT_ID';
const PROPERTY_INTERNAL_TOOL_ENDPOINT = 'INTERNAL_TOOL_ENDPOINT';
const PROPERTY_INTERNAL_TOOL_API_KEY = 'INTERNAL_TOOL_API_KEY';
```

### Step 2: Code.gs に関数追加

1. `getZoomHistory()` - Zoom履歴取得
2. `getGoogleMeetHistory()` - Google Meet履歴取得
3. `getInternalToolHistory()` - 社内ツール履歴取得
4. `getAllToolHistoryV5(dateString)` - V5統合取得

### Step 3: Index.html に設定チェックボックス追加

- Zoom
- Google Meet
- 社内ツール

---

## 完了条件

1. Zoom履歴が表示される
2. Google Meet履歴が表示される
3. 社内ツール履歴が表示される（設定時）
4. V4機能が正常動作する

---

## 注意事項

- Zoom Token設定が必要
- 社内ツールエンドポイント設定が必要
- Google MeetはカレンダーAPIから取得
