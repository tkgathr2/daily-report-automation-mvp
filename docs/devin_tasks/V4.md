## タスク概要

簡単日報くん V4（Docs/Drive/Dropbox連携）を実装する。

## 前提条件

- V3が完成・稼働していること

## 参照ドキュメント

- 要件定義書: `docs/specs/V4_requirements_ASK.md`
- 仕様書: `docs/specs/V4_plan.md`

---

## 実装手順

### Step 1: Code.gs に定数追加

```javascript
const PROPERTY_DROPBOX_TOKEN = 'DROPBOX_ACCESS_TOKEN';
```

### Step 2: Code.gs に関数追加

1. `getGoogleDocsHistory()` - Googleドキュメント履歴取得
2. `getGoogleDriveHistory()` - Google Drive履歴取得
3. `getDropboxHistory()` - Dropbox履歴取得
4. `getAllToolHistoryV4(dateString)` - V4統合取得

### Step 3: Index.html に設定チェックボックス追加

- Google Docs
- Google Drive
- Dropbox

---

## 完了条件

1. Googleドキュメント履歴が表示される
2. Google Drive履歴が表示される
3. Dropbox履歴が表示される
4. V3機能が正常動作する

## 注意事項

- Dropbox Access Token設定が必要
- GASのDriveApp/DocumentAppは自動認証
