## 4. 新テンプレート（V2で採用）

```
【YYYY年MM月DD日　氏名　日報】

【今日やったこと】
⚫︎
⚫︎
⚫︎

【わかった事・問題・共有事項】
⚫︎
⚫︎
⚫︎

【売上・利益に関わるポイント】
⚫︎
⚫︎
⚫︎

【次すること】
⚫︎
⚫︎
⚫︎
```

### 確定した仕様
- **氏名**: Googleアカウントから自動取得
- **日付形式**: YYYY年MM月DD日
- **全セクション**: 必須入力（空欄だとSlack送信不可）
- **翌日引き継ぎ**: 「次すること」を翌日に自動表示

## 5. Devinへの実装順序

```
V2 → V3 → V4 → V5 → V6 → V7
```

各バージョンは前のバージョンが完成していることが前提。

### 各バージョンの難易度と所要時間（目安）

| バージョン | 難易度 | 所要時間目安 | 必要な外部設定 |
|:----------:|:------:|:------------:|----------------|
| V2 | 低 | 2〜3時間 | なし |
| V3 | 高 | 4〜6時間 | Slack追加スコープ、Notion Token |
| V4 | 高 | 3〜4時間 | Dropbox Token |
| V5 | 中〜高 | 3〜4時間 | Zoom Token、社内ツールエンドポイント |
| V6 | 高 | 3〜4時間 | OpenAI API Key |
| V7 | 中 | 2〜3時間 | なし |

## 7. 注意事項

1. **V2が最優先**: まずV2を完成させてから次のバージョンに進む
2. **段階的実装**: 1バージョンずつ実装・テスト・完了を確認
3. **エラーハンドリング**: 各ツール連携は独立動作（1つ失敗しても他は動く）
4. **シークレット管理**: API KeyやTokenはScript Propertiesに保存
