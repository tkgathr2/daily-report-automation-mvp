# V004 Slack投稿先表示機能 仕様書

## 目的
Slack連携済みユーザーに対して、
どのチャンネルに日報が投稿されるかを事前に明示し、
誤投稿を防止する。

## 表示条件
- Slack連携（OAuth）が完了しているユーザーのみ表示する
- 未連携の場合は表示しない

## 表示内容
- 投稿先 Slack チャンネルID を表示する
- 表示するIDは Script Properties に保存されている
  SLACK_CHANNEL_ID の値と一致していること

## データソース
- Script Properties
  - Key: SLACK_CHANNEL_ID

## 表示場所
- 日報入力画面内
- 投稿ボタン付近にテキスト表示（UI詳細は実装側に委ねる）

## 変更対象ファイル
- src/Code.gs
- src/Index.html

## 非対象（今回やらないこと）
- チャンネル名の取得・表示
- チャンネル変更機能
- 複数チャンネル対応
- Slack API の追加呼び出し

## 受け入れ条件
- Slack連携済みの場合のみ表示される
- 表示されるIDが SLACK_CHANNEL_ID と一致する
- 既存機能（日付選択、投稿、完了画面）に影響がない
