## 実行ルール
1. 各テストの前にブラウザConsoleをクリアする
2. 失敗したケースは即座にスクショ＋Consoleログを保存する
3. alert/モーダルが出た場合は全文をコピペする
4. errorId/requestIdが表示されている場合は控える
5. Knowhow King（https://knowhow.up.railway.app）で該当errorIdを検索（可能なら）
6. 1日の実施上限は決めないが、集中力が落ちたら中断する
7. 本番に状態を残す操作（投稿等）はテスト用チャンネルで行うことを推奨
