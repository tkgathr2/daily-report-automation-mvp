# Claude For Chrome への指示：GASプロジェクトの整理（スクリプトID確認＋不要プロジェクト削除）

**バージョン**: V0.6
**作成日**: 2026-02-08

---

## これは Claude For Chrome への指示です。

---

## 目的

GAS上に「簡単日報くん」と「簡単日報くん v2」の2つのプロジェクトが存在しており、混乱の原因になっている。
本番プロジェクトを特定し、不要なほうを削除する。

---

## 手順（上から順に実施）

### Step 1: 「簡単日報くん v2」のスクリプトIDを確認

1. https://script.google.com/ を開く
2. **「簡単日報くん v2」** をクリックして開く
3. 左メニューの **「プロジェクトの設定」**（歯車アイコン）をクリック
4. **「スクリプト ID」** の値をコピー

### Step 2: 「簡単日報くん」のスクリプトIDを確認

1. プロジェクト一覧に戻る
2. **「簡単日報くん」** をクリックして開く
3. 左メニューの **「プロジェクトの設定」**（歯車アイコン）をクリック
4. **「スクリプト ID」** の値をコピー

### Step 3: 報告（ここで一旦停止）

以下の形式でチャットに報告してください：

```
簡単日報くん v2 のスクリプトID：（ここに貼る）
簡単日報くん のスクリプトID：（ここに貼る）
```

**照合対象**: `.clasp.json` の scriptId は `1BkBflFikzsT4yiQr6NWzCbWwcTkwKV2PmBqKlPgwBuO67es-2TVeC5AZ`

### Step 4: 不要プロジェクトの削除（報告後に実施）

- スクリプトIDが `.clasp.json` と **一致するほう** → 本番（残す）
- スクリプトIDが `.clasp.json` と **一致しないほう** → 不要（削除する）

削除手順：
1. 不要なプロジェクトを開く
2. 左メニューの「概要」を表示
3. 右上の「︙」（三点メニュー）をクリック
4. 「削除」を選択
5. 確認ダイアログで「削除」をクリック

### Step 5: API実行可能ファイルのデプロイ確認

本番プロジェクト（残したほう）で：
1. 「デプロイ」→「デプロイを管理」を開く
2. 「API実行可能ファイル」タイプのデプロイが存在するか確認
3. 存在しない場合 → 「新しいデプロイ」→「API実行可能ファイル」→「自分のみ」→デプロイ

### Step 6: GCPプロジェクト番号の確認

本番プロジェクトで：
1. 「プロジェクトの設定」を開く
2. 「Google Cloud Platform（GCP）プロジェクト」セクション
3. **プロジェクト番号**（数字のみ）をコピー

### 最終報告

```
本番プロジェクト名：○○○
不要プロジェクト：削除済み / まだ残っている
API実行可能ファイル：デプロイ済み / 未デプロイ
GCPプロジェクト番号：（数字を貼る）
```
