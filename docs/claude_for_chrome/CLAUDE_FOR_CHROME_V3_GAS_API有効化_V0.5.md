# Claude For Chrome への指示：GAS API実行可能ファイルのデプロイ

**バージョン**: V0.5
**作成日**: 2026-02-08

---

## これは Claude For Chrome への指示です。

---

## 目的

`clasp run` でGAS関数をローカルから直接実行できるようにする。
現在は「Webアプリ」形式のみデプロイされており、「API実行可能ファイル」が未デプロイのため失敗している。

---

## SEC-01

この工程ではシークレット情報が表示される可能性があります。SEC-01 で停止します。
API KeyやToken等が画面に表示された場合、チャット・記録に残さないこと。

---

## 手順（上から順に実施）

### Step 1: Google Cloud Console で Apps Script API を有効化

1. https://console.cloud.google.com/apis/api/script.googleapis.com を開く
2. 「有効にする」ボタンがあればクリック（既に有効なら次へ）
3. 「APIが有効です」と表示されることを確認

### Step 2: GASエディタで「API実行可能ファイル」としてデプロイ

1. https://script.google.com/ を開く
2. プロジェクト「簡単日報くん」（または `daily-report-automation-mvp`）を開く
3. 左メニューの「デプロイ」→「新しいデプロイ」をクリック
4. 種類の横の歯車アイコンをクリック
5. 「API実行可能ファイル」を選択
6. アクセスできるユーザー：「自分のみ」を選択
7. 「デプロイ」ボタンをクリック
8. デプロイIDが表示されることを確認

### Step 3: 完了確認

デプロイ完了後、以下だけ報告してください：

```
Step 1 (Apps Script API): OK/NG
Step 2 (API実行可能ファイル デプロイ): OK/NG
```

---

## 注意

- Webアプリのデプロイ（@43）はそのまま残す（削除しない）
- API実行可能ファイルは**別のデプロイ**として追加される
- SEC-01：デプロイIDやAPI Key等はチャットに貼らない
