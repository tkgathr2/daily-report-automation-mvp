## 確認手順（上から順に実施）

### 1) 画面表示
- 上記URLを開く
- 白画面にならず正常表示されるか確認

### 2) ツール設定UI
- 画面上部に **4つのチェックボックス**（Slack / Gmail / Gmail受信 / Notion）＋「設定を保存」ボタンが表示されるか確認
- Gmail受信が**デフォルトOFF**（チェックなし）か確認

### 3) 日付指定UI
- 年/月/日のセレクトボックスが表示されるか確認
- 前日/翌日ボタンが表示されるか確認
- 日付を変更して「履歴を取得」→カレンダー予定が変わるか確認

### 4) カレンダー自動反映
- 初期表示で当日のカレンダー予定が「今日やったこと」に入っているか確認

### 5) V2機能（後方互換）
- 4セクション全て未入力で「Slackに投稿」→エラーが出るか確認（バリデーション）
- 全セクションに入力して「コピー」→テンプレート形式でコピーされるか確認

### 6) Slack送信
- 全セクションに入力して「Slackに投稿」→送信成功するか確認
